# Discovery 1 — Cuestionario Operativo

## Propósito
Este cuestionario está diseñado para entrevistar a **Gonzalo Larroque** y relevar el funcionamiento real de Remistar, sin asumir ni inventar procesos, campos o reglas de negocio. Todo lo que Gonzalo no pueda responder en el momento debe registrarse como **pendiente** en el inventario de información ([`REMISTAR-DISCOVERY-2-inventario-informacion.md`](REMISTAR-DISCOVERY-2-inventario-informacion.md)).

## Cómo usar este documento
- Cada bloque temático tiene preguntas marcadas como **[CRÍTICA]** o **[secundaria]**.
  - **[CRÍTICA]**: sin esta respuesta no se puede avanzar con el diseño del proceso correspondiente.
  - **[secundaria]**: enriquece el diseño pero no bloquea el avance; puede quedar pendiente para una siguiente ronda.
- Siempre que sea posible, pedir a Gonzalo **ejemplos reales**: capturas de WhatsApp, planillas de Excel, remitos, facturas, cuadernos, mensajes tipo, fotos de vencimientos, etc. Un ejemplo real vale más que una descripción abstracta.
- Las respuestas deben registrarse tal cual las da Gonzalo, sin reinterpretarlas ni "prolijarlas" en esta etapa.
- Si una respuesta genera una nueva pregunta no prevista aquí, agregarla al final de la sección correspondiente.
